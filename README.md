# 🚀 European Football Transfer & Squad Analyzer

Ein datengetriebenes Portfolio-Projekt zur Analyse von Transfermärkten und Kaderstrukturen im europäischen Fußball. Das Projekt dient als Vorbereitung auf meine Ausbildung zum Fachinformatiker für Daten- und Prozessanalyse[cite: 1].

---

## 📌 Projektüberblick
* **Ziel:** Entwicklung einer End-to-End-Pipeline, die Fußball-Statistiken bereinigt, in einer relationalen Datenbank strukturiert und in einem interaktiven Power BI Dashboard für Scouting-Entscheidungen visualisiert[cite: 1].
* **Zeitraum:** Juli – September 2026 (~11 Wochen)[cite: 1]
* **Tech Stack:** Python (Pandas), SQL (SQLite / DBeaver), Power BI, BPMN (Draw.io)[cite: 1]

---

## 📁 Projektstruktur
Hier ist dokumentiert, wo welche Projektschritte zu finden sind:
* `/data` - Enthält unberührte Rohdaten und die final bereinigten Datensätze.
* `/notebooks` - Python-Notebooks für Data Cleaning und Feature Engineering (Per-90-Stats)[cite: 1].
* `/sql` - Datenbank-Modelle, Import-Skripte und Filter-Queries[cite: 1].
* `/dashboard` - Die Power BI Dashboard-Datei (`.pbix`) samt Scoring-Modell[cite: 1].
* `/docs` - Prozessdokumentation (Soll-/Ist-Prozesse als BPMN)[cite: 1].
